# sistema-parque-estacionamento
Este projeto consiste no desenvolvimento de um sistema de gestão de parque de estacionamento, com o objetivo de automatizar o controle de entrada e saída de veículos, melhorar a organização e otimizar a gestão de vagas.  A aplicação permite o registo de veículos, controlo de tempo de permanência, cálculo automático de tarifas e geração de relatório
